# Vrinda Store Annual Sales Analysis 2022

A complete sales data analysis for Vrinda Store, an Indian fashion e-commerce brand. This project explores order trends, customer demographics, top-performing states, and sales channel performance across 2022.

---

## 📊 Project Overview

The dataset contains **31,000+ orders** across multiple online platforms throughout 2022. The analysis covers:

- Monthly order volume and revenue trends
- Sales breakdown by gender and age group
- Top 5 states by revenue
- Channel-wise order distribution
- Order status summary (delivered, returned, cancelled, refunded)

---

## 📁 File Structure

```
Vrinda_Store_Data_Analysis.xlsx
├── Dashboard          → Visual summary of all key metrics
├── Vrinda Store       → Raw order-level data (~31,000 rows)
├── order vs sales     → Monthly orders and revenue
├── men                → Revenue split by gender
├── age vs gender      → Order share by age group and gender
├── states             → Top 5 states by revenue
├── Channels           → Order share by sales platform
└── Status             → Order count by delivery status
```

---

## 🔍 Key Insights

- **Amazon dominates** with ~35% of all orders, followed by Myntra (~23%) and Flipkart (~22%)
- **Women drive 64%** of total sales; men account for 36%
- **Adult women** (25–45 age group) are the largest customer segment
- **Maharashtra** is the top revenue state (₹29.9L), followed by Karnataka (₹26.5L)
- **Peak months** were March and February; orders declined steadily from April onwards
- **93% of orders** were successfully delivered; returns and cancellations together account for ~6%
